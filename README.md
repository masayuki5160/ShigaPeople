# Shiga

## How to build

```
# Install expo cli
$ npm install expo -g expo-cli

# Start Expo Server and use QR code with Expo Go App
$ npm start
```

## Archive for Submission

```
# Check expo account
$ expo whoami

# App Store
$ expo build:ios

# Google Play
$ expo build:android
```
